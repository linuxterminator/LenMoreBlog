在我学习`springboot`的过程中我遇到了很多困难，希望我的学习过程可以给同为学习者的你提供方向和参考

### 构建
***
```shell
#克隆项目
git clone 
#安装依赖
mvn install
#构建
mvn package
```
### 运行
***
```shell
#使用mvn运行
mvn run:springboot
#使用jar运行
java -jar target/LenMore-0.0.1-SNAPSHOT.jar
```
